# World phone

Carry a world phone and open it from the command palette. In-world extension or user dialing is the primary phone path. Calling an outside phone number is optional convenience dialing through FlexPBX.

FlexPBX outbound dialing can be enabled in App settings, where you can enter one or more prefix digits (default: `9`). These preferences do not grant authorization or bypass account checks: the server must verify the account and explicitly allow outbound dialing before an outside number can be connected.
