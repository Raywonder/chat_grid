# Device Button Sound Pack

Generated with ElevenLabs sound effects on 2026-07-14 for short, tactile device interactions.

Use these as one-shot sounds for radios, appliances, keypads, switches, and similar interactable items.

## Files

- `radio_power.mp3` - small vintage radio power button press, tactile plastic click with spring return.
- `radio_tuner_step.mp3` - compact radio tuning button single step, crisp plastic tick with faint relay feel.
- `preset_button.mp3` - small rubber radio preset button press, softer click and release.
- `keypad_tactile.mp3` - firm accessible-device keypad click, no beep.
- `soft_plastic_press.mp3` - muted appliance-style plastic button press.
- `hardware_toggle.mp3` - small hardware toggle switch flick with light metal snap.

All files are 0.8 second MP3 one-shots at 44.1 kHz / 128 kbps.
